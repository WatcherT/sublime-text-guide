# iTg


[github项目地址](https://github.com/itsthatguy/theme-itg-flat)

安装方法：
```
shift + cmd + p 打开命令面板
输入 “Package Control: Install Package” 命令
输入Theme - itg.flat, 找到后回车安装
安装成功后在preferences ->Color Scheme中选择主题
```
